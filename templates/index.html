Projetos

Projeto: Tabela Fipe - API

O projeto Tabela Fipe é caracterizado por possuir um sistema de consulta da tabela fipe, tabela na qual expressa o preço médio dos veículos dentro do mercado nacional, por meio do consumo da api pública tabela fipe e a utilização de linguagem de programação java a fim de relatar os preços médios da maioria das categorias veiculares do mercado nacional, como motocicletas, carros e caminhões.

Linguagem usada:

JAVA

Projeto: Trading com Python

O projeto Trading com Python tem como objetivo mostrar os momentos corretos de venda e compra de ações a partir do mercado financeiro nacional a partir das bibliotecas yfinance, pandas e numpy nas quais possibilita a busca de ações, informações e geração de estatísticas e tabelas sobre a mesma. De maneira geral, esta permite o estudo de determinada ação de maneira completa e de fácil compreensão, mas a partir de conhecimentos gerais sobre o mercado de ações.

Linguagem usada:

PYTHON

Projeto: Screen Match

O projeto Screen Match também foi realizado durante o período de estudo dentro da plataforma Alura. Tal projeto possui como objetivo o estudo da linguagem de programação java e determinadas características. Ademais, o projeto é caracterizado por criar um site, voltada para o back-end, clone ao streamings a partir do consumo da api Omdb. Esta aplicação é capaz de buscar qualquer filme ou série atuais, além de relatar quantidade de episódios, duração, dentre outras funções.

Linguagem usada:

JAVA
